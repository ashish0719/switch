# Switch Project

This is a comprehensive full-stack application featuring:
- **Backend**: Node.js, Express, MongoDB, Cloudflare R2 for storage.
- **Frontend**: React, Vite, Tailwind CSS.
- **Features**: User Authentication, Resume Optimization, Notes/Resource Sharing (Admin Uploads).

## Setup

1.  **Backend**:
    -   `cd Backend`
    -   `npm install`
    -   Create `.env` file with credentials.
    -   `npm start`

2.  **Frontend**:
    -   `cd Frontend`
    -   `npm install`
    -   `npm run dev`
